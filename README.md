## 简易图书馆管理系统
此图书馆管理系统能实现用户检索图书、查看自己的借阅信息和预约图书，管理员管理图书和借书还书功能，建立一个方便的图书馆信息交互平台。
## 功能介绍
用户注册登录
管理员对图书进行增删查改
管理员操作借书还书
用户个人书架
用户预约借书
检索图书
## 运行环境

客户端系统:Windows XP professional/win10 旗舰版
